# KeyAura

Your Colored Keyboard - A professional Android custom keyboard with stunning radial color animations.

## Features
- Multi-color radial animations on every key press
- Roman Urdu to Urdu transliteration
- Smart Enter key with context-aware actions
- Auto-return to letters after symbol/number input
- Customizable settings (vibration, sound, backspace speed)
- Landscape mode support with fixed height
- Beautiful dark UI

## Build
```bash
./gradlew assembleDebug
```

## Install
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```
